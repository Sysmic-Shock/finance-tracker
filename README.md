# finance-tracker
A CLI-based personal finance tracker built in python, designed to help users log  financial transactions. The purpose of this is to create a functional and reliable transaction history that can be used to highlight trends so users can make informed financial decisions.

## Features
You will be able to:
- Insert a transaction to track incoming and outgoing finances
- Modify details of any transactions entered ensuring any mistakes can be corrected
- Remove any transaction currently shown in your financial history
- View clear and reliable feedback whenever a transaction is inserted, modified or removed to ensure your data is exactly the way YOU want it
- Assign categories to each transaction and filter by those categories
- View history of all transactions made.
- Construct a simple password in order to provide security
- View visual representation of data via analytics to provide you with as many ways to view your finances as possible
- Make data persistent across multiple sessions so you can be confident that your data will be saved when you exit or close the application

While this will developed to be a CLI-based application, as it grows it may change into something much more visually appealing.
